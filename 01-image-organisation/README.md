# image-organisation

## Coming soon

This stage will cover how I organise raw camera trap images before any AI processing.

It will include Python scripts that:
- sort images into a consistent folder structure, and
- renames image files based on their folder path, so every file ends up with a unique, traceable name.

The aim is a tidy, predictable starting point that the later stages can rely on. Code and a short how-to guide will be added here soon.
